# 📊 Data Analytics & Visualization Projects

## 📌 Project Overview
This repository contains two data visualization projects built using Power BI and Tableau. The projects focus on analyzing business data and presenting insights through interactive dashboards.

## 🔷 Project 1: Customer Segmentation & Sales Analysis (Power BI)

### 🎯 Objective
Analyze sales data to identify customer segments and understand purchasing behavior.

### 📂 Dataset
- Customer ID
- Name
- Age
- Region
- Product
- Category
- Sales Amount
- Purchase Date

### 🛠️ Tasks Performed
- Data cleaning and transformation in Power BI
- Created calculated columns and measures:
  - Total Sales per Customer
  - Average Purchase Value
  - Customer Age Groups (18–25, 26–35, etc.)

### 📊 Dashboard Features
- Bar Chart: Sales by Customer Segment (Age Group & Region)
- Pie Chart: Product Category Distribution
- Line Chart: Monthly Sales Trend
- Table: Top 10 Customers by Sales
- Slicers: Region, Age Group, Product Category
- Conditional Formatting for high-value customers

### 📈 Key Insights
- Identified most valuable customer segments
- Analyzed purchasing trends across regions
- Detected high-revenue customers for targeted marketing

## 🔷 Project 2: Movie Box Office Dashboard (Tableau)

### 🎯 Objective
Visualize movie performance and box office trends.

### 📂 Dataset
- Movie Name
- Genre
- Release Year
- Budget
- Revenue
- Rating

### 📊 Dashboard Features
- Bar Chart: Revenue by Genre
- Line Chart: Box Office Trends Over Years
- Scatter Plot: Budget vs Revenue
- Top Movies by Revenue
- Filters for Genre and Year

### 📈 Key Insights
- Identified top-performing genres
- Compared budget vs revenue efficiency
- Analyzed yearly trends in movie performance

- Excel / CSV
